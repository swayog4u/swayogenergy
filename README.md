# Swayog

This repository uses Node.js and TypeScript.

## New dependencies added

- nodemailer (production): SMTP/email sending library.
- @types/nodemailer (dev): TypeScript types for nodemailer.
- tsx (dev): Tool to run TypeScript files directly during development.

## Installation

Run the following to install the new dependencies:

```bash
npm install nodemailer
npm install -D @types/nodemailer tsx
```

Also install project dependencies if needed:

```bash
npm install
```
